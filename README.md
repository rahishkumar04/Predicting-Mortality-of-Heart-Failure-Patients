A machine learning project that predicts the risk of heart failure using clinical data.
It includes data preprocessing, exploratory data analysis (EDA), and model building using algorithms like Logistic Regression, KNN, Random Forest, and XGBoost.

Features:
Data cleaning & visualization
Multiple ML models
Performance evaluation (Accuracy, Precision, Recall, F1, ROC-AUC)

Tech Stack: Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn, XGBoost
